# Random Password Generator

**Автор:** Артем Назаров

## Описание

Random Password Generator — это приложение с графическим интерфейсом для генерации безопасных случайных паролей. Пользователь может настроить длину пароля и выбрать, какие символы использовать: цифры, буквы или спецсимволы. Все сгенерированные пароли сохраняются в историю с датой создания, которую можно просматривать и очищать.

## Возможности

- Настройка длины пароля (от 4 до 32 символов)
- Выбор типов символов:
  - Цифры (0-9)
  - Буквы (A-Z, a-z)
  - Специальные символы (!@#$%^&* и др.)
- Генерация случайного пароля одним кликом
- Копирование пароля в буфер обмена
- Сохранение истории паролей в JSON-файл
- Просмотр и очистка истории
- Проверка корректности ввода (минимальная длина, выбор символов)
